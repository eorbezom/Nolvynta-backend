# Nolvynta - Backend E-commerce

Backend completo para la tienda en línea **Nolvynta**, desarrollado con **Spring Boot 4.0.6 + Java 21 + PostgreSQL**.

## 🚀 Tecnologías utilizadas

- **Java 21**
- **Spring Boot 4.0.6**
- **Spring Data JPA**
- **PostgreSQL**
- **Spring Security + JWT** (en desarrollo)
- **Lombok**
- **Swagger / OpenAPI**
- **Maven**

## 📋 Requisitos

- Java 21
- PostgreSQL 16 o superior
- Maven 3.8+
